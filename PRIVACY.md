# Privacy Policy

_Last updated: 2026-07-14_

Trust Issues is a local, read-only security-review skill for Claude Code, Claude
Desktop/Cowork, Codex, and compatible agents. It runs on the user's own machine,
invoked by the AI agent.

## What we collect

Nothing. Trust Issues does not collect, store, transmit, or share any personal data,
usage data, telemetry, or the contents of any code it reviews. There is no backend
server, no account, and no analytics.

## How it works

Trust Issues reads files locally to produce a security assessment. It never executes
the code it reviews and never sends your files or its findings anywhere. Any network
access is initiated by you or your agent — for example cloning a public repository you
asked it to review, or the agent's own web search — not by this skill collecting data.

## Third parties

Trust Issues integrates no third-party data processors, trackers, or advertising.
